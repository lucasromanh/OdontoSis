# Design System: Dental Pro (Light Aesthetic)

## Visual Theme
The application follows a "Med-Tech Modern" aesthetic: clean, professional, and accessible. It uses a light color palette with soft blue backgrounds and high-contrast accents for clinical clarity.

## Color Palette
- **Primary Blue**: `#137FEC` (Professional, trustworthy)
- **Background**: `#F0F7FF` (Soft clinical blue-gray)
- **Surface/Cards**: `#FFFFFF` (Pure white for focus)
- **Typography (Heading)**: `#1E293B` (Deep slate)
- **Typography (Body)**: `#64748B` (Slate medium)
- **Accents**: 
  - Emerald: `#10B981` (Completed, Success)
  - Orange: `#F59E0B` (Pending, Attention)
  - Red: `#EF4444` (Critical, Urgent)

## Typography
- **Font Family**: Inter (Standard for modern web apps)
- **Headings**: Semibold/Black, often Italic for a "Pro" feel.
- **Tracking**: Tight for headings (`tracking-tighter`), extra wide for labels (`tracking-widest`).

## Component Styling
- **Border Radius**: `2rem` (32px) for main containers and cards. `1rem` (16px) for inner elements.
- **Shadows**: Soft, layered shadows (`shadow-sm`) with colored glows for primary elements (`shadow-blue-500/20`).
- **Cards**: Pure white background with very light borders (`border-slate-200/60`).

## Layout Principles
- **Sidebar**: Fixed width (64/20), white background, active item uses Primary Blue.
- **Header**: Glassmorphism (`backdrop-blur-xl`) with white opacity (70%).
- **Whitespace**: Generous padding (`p-8` to `p-12`) to allow high information density without clutter.
